# Course Overview

This course provides a deep dive into Model Context Protocol (MCP), a new open standard transforming how LLM-powered agents interact with external systems.

## What will you learn?

The course will provide info about architecture of agentic systems built with MCP, introducing the concepts of host, client, and server.

## Course Structure

Concept of MCP:

- Limitations of LLMs
- The transition to agentic systems
- The problem MCP was designed to solve
